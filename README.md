# Typescript backend service template

- Node 10
- Typescript 3
- Jest

---

## The Aim

Quick start into a typscript template.

## NPM Commands

| Command      | Purpose                                                                                |
| ------------ | :------------------------------------------------------------------------------------- |
| `build`      | Builds the project                                                                     |
| `lint`       | Checks the code style                                                                  |
| `lint:fix`   | Tries to fix linter errors in the project                                              |
| `watch`      | Watches source files, rebuilds, lint's and restarts the server if changes are detected |
| `test`       | Runs all tests and reports code coverage                                               |
| `watch-test` | Watches source files and reruns tests if changes are detected                          |
